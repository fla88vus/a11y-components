# Contributing

Thank you for your interest in contributing to a11y-ui-kit-react!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/a11y-components.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
# Start development
npm run dev

# Run tests
npm test

# Run Storybook
npm run storybook
```

## Pull Request Process

1. Update documentation for any new features
2. Add tests for new functionality
3. Ensure all tests pass
4. Follow the existing code style
5. Create a pull request with a clear description

## Accessibility Standards

All components must meet WCAG 2.1 AA standards. Test with:
- Screen readers (NVDA, JAWS, VoiceOver)
- Keyboard navigation only
- jest-axe automated tests

For more details, see the full [CONTRIBUTING.md](https://github.com/fla88vus/a11y-components/blob/main/CONTRIBUTING.md) in the repository.
